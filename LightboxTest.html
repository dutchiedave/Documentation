DBootstrap

The DBootstrap package provides all the functionality to launch a game with functions to load and close scenes using Unity's Addressable package.

How to use

1. Drag the 'Bootstrap' prefab from the Prefabs folder into the Scene. 2. Create a SceneData Scriptable Object in the Project folder by Right Clicking and selecting 'DBootStrap -> SceneData. 3. Add the created SceneData to the Bootstrap Component in the Scene.
